Slip 4 
Write a program to access online weather API.

Weather.php
<?php

// Replace with your API key from OpenWeatherMap
$apiKey = "YOUR_API_KEY";

// City name
$city = "Pune";

// API URL
$url = "https://api.openweathermap.org/data/2.5/weather?q=$city&appid=$apiKey&units=metric";

// Fetch data from API
$response = file_get_contents($url);

// Convert JSON to PHP array
$data = json_decode($response, true);

// Display weather information
if ($data && $data["cod"] == 200) {
    echo "City: " . $data["name"] . "<br>";
    echo "Temperature: " . $data["main"]["temp"] . " °C<br>";
    echo "Humidity: " . $data["main"]["humidity"] . "%<br>";
    echo "Weather: " . $data["weather"][0]["description"] . "<br>";
} else {
    echo "Error fetching weather data";
}
?>
How to Get API Key
1.	Go to: https://openweathermap.org/api
2.	Sign up (free)
3.	Copy your API Key
4.	Replace "YOUR_API_KEY" in code

        function checkPassword() {
            var username = document.getElementById("user").value;
            var password = document.getElementById("pass").value;

            // Conditions
            var hasNumber = /[0-9]/.test(password);
            var hasAlpha = /[a-zA-Z]/.test(password);
            var hasSpecial = /[!@#$%^&*]/.test(password);
            var lengthCheck = password.length > 8;

            if (hasNumber && hasAlpha && hasSpecial && lengthCheck) {
                document.getElementById("result").innerHTML = "You entered a strong password";

                document.getElementById("result").innerHTML = "Enter complex password";

        }
