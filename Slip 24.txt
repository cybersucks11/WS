 Q.2 Write a program that uses a currency exchange API (e.g., Exchange Rate-API or Fixer.io) to perform the following:
 ● Convert a specified amount from one currency to another.
 ● Display the exchange rate for a given currency pair.
 ● Handle invalid currency codes or API errors gracefully.

ExchangeRate.php
<?php
// Replace with your API key
$apiKey = "YOUR_API_KEY";
// Input values
$from = "USD";
$to = "INR";
$amount = 100;
// API URL
$url = "https://v6.exchangerate-api.com/v6/$apiKey/latest/$from";
// Fetch API response
$response = @file_get_contents($url);
if ($response === FALSE) {
    die("Error: Unable to connect to API");
}

// Decode JSON
$data = json_decode($response, true);
// Check API status
if ($data["result"] != "success") {
    die("Error: Invalid API response");
}
// Check if target currency exists
if (!isset($data["conversion_rates"][$to])) {
    die("Error: Invalid currency code");
}
// Get exchange rate
$rate = $data["conversion_rates"][$to];
// Convert amount
$converted = $amount * $rate;
// Display results
echo "From Currency: $from <br>";
echo "To Currency: $to <br>";
echo "Exchange Rate: 1 $from = $rate $to <br>";
echo "Converted Amount: $amount $from = $converted $to <br>";
?>

How to Get API Key
1.	Visit: https://www.exchangerate-api.com/
2.	Sign up (free)
3.	Replace "YOUR_API_KEY" in code
