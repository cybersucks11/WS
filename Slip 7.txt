Q1] Create the following relations : Emp(eno,name,dno,salary) Project(pno,pname,control dno,budget) Each employee can work on one or more projects, and a project can have many employees working in it. The number of hours worked on each project by an employee also needs to be stored. 
1.	list the names of Project names whose budget is greater than 10000. 
2.	. list the names of projects, controlled by department No 101. 
3.	. list the details of the projects with second maximum budget 
4.	. list the details of the projects with maximum budget. 
5.	. list the names of employees, working on E&TC department
-- Employee Table
CREATE TABLE Emp ( eno INT PRIMARY KEY, name VARCHAR(50),dno INT, salary INT);

-- Project Table
CREATE TABLE Project ( pno INT PRIMARY KEY, pname VARCHAR(50),control_dno INT, budget INT);

-- Relationship Table (Employee works on Project)
CREATE TABLE Works_On (eno INT,pno INT, hours INT,PRIMARY KEY (eno, pno),FOREIGN KEY (eno) REFERENCES Emp(eno),FOREIGN KEY (pno) REFERENCES Project(pno));

Quries
1. Project names with budget > 10000
SELECT pname 
FROM Project 
WHERE budget > 10000;
2. Projects controlled by department 101
SELECT pname 
FROM Project 
WHERE control_dno = 101;
3. Project with Second Maximum Budget
SELECT * 
FROM Project 
WHERE budget = (
    SELECT MAX(budget) 
    FROM Project 
    WHERE budget < (SELECT MAX(budget) FROM Project));
 4. Project with Maximum Budget
SELECT * 
FROM Project 
WHERE budget = (SELECT MAX(budget) FROM Project);
 5. Employees working in E&TC department
Assuming E&TC department = dno = 101
SELECT name 
FROM Emp 
WHERE dno = 101;


Q2] Write a program to access online weather API.

Weather.php
<?php

// Replace with your API key from OpenWeatherMap
$apiKey = "YOUR_API_KEY";

// City name
$city = "Pune";

// API URL
$url = "https://api.openweathermap.org/data/2.5/weather?q=$city&appid=$apiKey&units=metric";

// Fetch data from API
$response = file_get_contents($url);

// Convert JSON to PHP array
$data = json_decode($response, true);

// Display weather information
if ($data && $data["cod"] == 200) {
    echo "City: " . $data["name"] . "<br>";
    echo "Temperature: " . $data["main"]["temp"] . " °C<br>";
    echo "Humidity: " . $data["main"]["humidity"] . "%<br>";
    echo "Weather: " . $data["weather"][0]["description"] . "<br>";
} else {
    echo "Error fetching weather data";
}
?>
How to Get API Key
1.	Go to: https://openweathermap.org/api
2.	Sign up (free)
3.	Copy your API Key
4.	Replace "YOUR_API_KEY" in code
