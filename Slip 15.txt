Q 1. Write a short report explaining: a) The identified threat. b) How the threat was exploited. c) The steps you took to mitigate the vulnerability. 

a) Identified Threat
The identified threat is SQL Injection.
It is a type of web security vulnerability where an attacker inserts malicious SQL queries through input fields (like login forms) to access or manipulate database data.
________________________________________
b) How the Threat was Exploited
The application was accepting user input without proper validation.
The attacker entered special SQL input instead of a normal password.
Example (concept):
•	Instead of a password, attacker enters a condition that is always true 
•	The database executes the modified query 
•	As a result, login authentication is bypassed 
________________________________________
c) Steps Taken to Mitigate the Vulnerability
1.	Used Prepared Statements (Parameterized Queries) 
2.	Applied input validation to restrict unwanted characters 
3.	Used password encryption for secure storage 
4.	Limited login attempts to prevent brute force attacks 
5.	Enabled proper error handling and logging
